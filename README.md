CyberShield | Advanced Threat Intelligence Landing Page

Project Overview
CyberShield is a high-fidelity, static web landing page built as part of a Week 1 Internship Task. The project involves transforming a conceptual wireframe into a fully functional, responsive website using modern HTML5 and CSS3. The design follows a "Security Operations Center" (SOC) aesthetic, prioritizing high contrast and technical clarity.

Features
Responsive Design: Fully optimized for Desktop, Tablet, and Mobile views using CSS Media Queries and Fluid Grids.
Semantic HTML: Built using structural tags (<nav>, <header>, <section>, <footer>) for better SEO and accessibility.
Smooth Navigation: Internal page linking with smooth-scroll behavior for an enhanced user experience.
Modern UI: Features a dark-themed "Midnight & Neon" interface with hover effects and glassmorphism-inspired elements.

Technologies Used
HTML5: For page structure and content hierarchy.
CSS3: For styling, Flexbox navigation, and CSS Grid layouts.
VS Code: Primary code editor used for development.

Project Structure
index.html: The main markup file containing the site structure.
style.css: The stylesheet containing all layout rules, theme variables, and responsiveness logic.

Layout Strategy
Navigation: Implemented using Flexbox to ensure a balanced distribution between the branding and menu items.
Content Grid: The "Core Capabilities" section utilizes CSS Grid with an auto-fit property, allowing the layout to adjust automatically to the screen size without breaking.
Mobile Optimization: A breakpoint at 768px triggers a layout shift, stacking elements vertically to ensure readability on smaller devices.
